Button and links

Primary button

Creating a primary button style, requires a combination of the following classes: .btn and .btn-primary.
The above classes can be applied to <button> or link <a> elements.

Standard buttons using the class .btn.primary
Icons inside of buttons can only appear to the right, and left padding is applied to the icon. As a result there should be no whitespace between the button text and the icon span tag
Square buttons can be made with the modifier .btn-square and have fixed dimensions. Ideal for icon buttons.

Default button

Standard buttons using the class .btn-default

Alternate button

Light buttons to be used against colorful backgrounds are an alternate to the primary button

Button Sizes

Buttons can appear in variety of defined sizes.

Disabled Buttons

Adding the attribute disabled to a button will prevent it from being clicked and provide custom styling

Button links

Buttons can be styled so that they appear to look more like standard links using the class .btn-link
Button links with icons can only appear on the left. There should be no whitespace after the icons span tag.
