Pagination

This is the default pagination style, used throughout the site. As well as being easily scalable, each link item has been created to have a large click area, for improved usability. To disable the prime pagination links (previous and next), simply toggle the .disabled CSS class. To mark the current page, corresponding with a list item, apply the .active CSS class to the list item.
